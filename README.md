# ✨ My June Glow-Up Plan ✨

A beautiful, mobile-friendly website for tracking your June glow-up journey.

## Features

- **Daily Habits Tracker** — 10 healthy habits (skincare, hydration, exercise, sleep & more) with tap-to-complete checkboxes
- **June Calendar** — visual calendar where you can mark your "glow days"
- **Streak Counter** — tracks how many consecutive days you've been consistent
- **Progress Bar** — shows how far through your 30-day June plan you are
- **Goals List** — editable list of personal June goals with checkboxes
- **Daily Affirmation** — rotating positive affirmations with a refresh button
- **Daily Quote** — a new motivational quote each day
- **Glow-Up Journal** — a text area to write reflections and wins, saved locally
- **100% local** — all data is saved in your browser's local storage (no account needed)

## How to use

### Option A — Open directly on your phone
1. Download or copy `index.html` to your phone
2. Open it in Safari (iPhone) or Chrome (Android)
3. Tap **Share → Add to Home Screen** to pin it as an app icon 📱

### Option B — Deploy on Vercel (recommended)
1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **Add New → Project** and import this repo (`computerdev21/vdGlowUp`)
3. Leave all settings as default — Vercel will detect the static site automatically
4. Click **Deploy** ✅
5. Your site will be live at `https://vd-glow-up.vercel.app` (or similar)
6. Share the link with your girlfriend 💖

> Every time you push to `main`, Vercel re-deploys automatically.

## Tech stack

Plain HTML + CSS + JavaScript — no frameworks, no build step, no dependencies.
